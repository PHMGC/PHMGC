## About Me

- Undergraduate researcher at [**VeRLab**](https://www.verlab.dcc.ufmg.br/) (Computer Vision & Robotics Lab, UFMG)
- Currently researching the integration of **Mixture of Experts** (MoE) into **Structure from Motion** (SfM) pipelines to improve 3D
reconstruction scalability, leveraging ACCV-published methodologies. 
- Learning more about **NLP**, **causal inference**, and **distributed systems**
- Technologies I work with: **Python**, **PyTorch**, **scikit-learn** and more
- Fun fact: I started college studying **State Science** before switching to CS
